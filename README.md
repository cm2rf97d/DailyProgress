# DailyProgress

由於身邊的家人朋友通常今天會很有熱情的預想明天該做的事情，不過到了隔天熱情卻消失了，通常也就忘記或懶惰不去做這件事，因此  
以目前所理解的MVVM架構，實作出一個幫助人們紀錄自己每天該完成項目的APP，在日期中輸入當天該完成的項目及項目內容，  
當天完成該項目後可按下完成按鈕，紀錄自己每天是否有照著自己的進度完成每天該完成的事項。
